1. **O que é um array em programação?**
   Um array é uma estrutura de dados que armazena vários valores em uma única variável.

2. **Por que é melhor usar um array do que criar várias variáveis separadas, como aluno1, aluno2 e aluno3?**
   Porque facilita a organização, o acesso e a manipulação dos dados, além de deixar o código mais simples.

3. **O que é índice em um array?**
 É a posição de cada elemento dentro do array.

4. **Por que o primeiro elemento de um array fica na posição 0?**
   Porque a maioria das linguagens de programação utiliza indexação iniciando em zero.

5. **Para que serve o `.length`?**
   Serve para informar a quantidade de elementos de um array.

6. **Como acessamos o primeiro elemento de um array chamado alunos?**
   Utilizando o índice 0 do array.

7. **Como podemos alterar um valor dentro de um array?**
   Acessando a posição desejada e atribuindo um novo valor a ela.

8. **Qual a relação entre arrays e laços de repetição?**
   Os laços de repetição permitem percorrer e processar todos os elementos de um array.

9. **Qual a diferença entre percorrer um array com `for` e apenas mostrar um item específico?**
   Percorrer um array acessa todos os seus elementos, enquanto mostrar um item específico acessa apenas uma posição.

10. **O que é um acumulador?**
    É uma variável utilizada para armazenar e atualizar resultados ao longo da execução do programa.

11. **Para que servem operadores aritméticos como `+`, `-`, `*` e `/` dentro de um programa?**
    Servem para realizar cálculos matemáticos.

12. **Para que servem operadores lógicos como `&&` e `||`?**
    Servem para combinar e avaliar condições lógicas.

13. **Qual a diferença entre `if/else` e `switch`?**
    O `if/else` é mais flexível para condições variadas, enquanto o `switch` é mais indicado para comparar um mesmo valor com várias opções.

14. **O que os métodos `push` e `pop` fazem em um array?**
    O `push` adiciona um elemento ao final do array e o `pop` remove o último elemento.

15. **O que é uma matriz e como ela se relaciona com arrays?**
    Uma matriz é uma estrutura formada por vários arrays organizados em linhas e colunas.
