
1. O que são operadores em programação?
Operadores são símbolos usados para realizar ações com valores e variáveis. Eles servem para fazer contas, comparações e outras operações dentro do código.

2. Para que servem os operadores aritméticos? Cite exemplos.
Os operadores aritméticos são usados para fazer cálculos matemáticos. Alguns exemplos são +, -, *, / e %.

3. Qual a diferença entre / e % em JavaScript?
O operador / faz a divisão entre dois números. Já o % mostra o resto da divisão.

4. Como descobrir se um número é par usando o operador %?
Basta verificar se o resto da divisão por 2 é igual a 0. Se for, o número é par.

5. O que é uma expressão em programação?
Uma expressão é uma combinação de valores, variáveis e operadores que gera um resultado.

6. O que significa precedência de operadores?
É a ordem em que as operações são executadas em uma expressão. Algumas operações acontecem antes de outras.

7. Por que os parênteses podem mudar o resultado de uma expressão?
Porque tudo que está dentro dos parênteses é executado primeiro. Isso pode alterar o resultado final da conta.

8. Qual a diferença entre =, == e ===?
O = serve para atribuir um valor. O == compara valores e o === compara valores e tipos ao mesmo tempo.

9. Por que é recomendado usar === em JavaScript?
Porque ele faz uma comparação mais precisa, evitando erros causados pela conversão automática de tipos.

10. O que os operadores relacionais retornam: texto, número ou boolean?
Eles retornam um valor booleano, ou seja, true ou false.

11. Quando usamos o operador >? E o operador >=?
O > é usado quando queremos saber se um valor é maior que outro. O >= verifica se ele é maior ou igual.

12. Quando usamos o operador <? E o operador <=?
O < verifica se um valor é menor que outro. Já o <= verifica se ele é menor ou igual.

13. Para que serve o operador lógico &&?
Ele representa o "E". As duas condições precisam ser verdadeiras para o resultado ser verdadeiro.

14. Para que serve o operador lógico ||?
Ele representa o "OU". Basta uma das condições ser verdadeira para o resultado também ser verdadeiro.

15. Para que serve o operador lógico !?
Ele serve para inverter um valor lógico. Se for true, vira false, e vice-versa.

16. Qual a diferença entre uma condição simples e uma condição composta?
A condição simples possui apenas uma comparação. A composta possui duas ou mais condições ligadas por operadores lógicos.

17. Explique uma situação real em que duas condições precisam ser verdadeiras ao mesmo tempo.
Por exemplo, para entrar em uma prova é necessário estar inscrito e apresentar um documento. As duas condições precisam ser verdadeiras.

18. Explique uma situação real em que apenas uma entre duas condições precisa ser verdadeira.
Por exemplo, para entrar em um evento a pessoa pode apresentar RG ou CNH. Basta um dos documentos.

19. Como o console.log() pode ajudar a encontrar erros no código?
Ele permite mostrar informações na tela durante a execução do programa. Assim fica mais fácil descobrir onde está o erro.

20. Por que devemos testar o mesmo programa com valores diferentes?
Porque cada valor pode gerar um resultado diferente. Isso ajuda a garantir que o programa funcione corretamente em várias situações.
