1. O que é programação?
Programação é o processo de criar instruções para que um computador execute determinadas tarefas. Ela é usada para desenvolver sites, aplicativos, sistemas e diversas outras tecnologias.

2. O que é um algoritmo?
Um algoritmo é uma sequência de passos organizados para resolver um problema ou atingir um objetivo. Ele funciona como um roteiro que indica o que deve ser feito.

3. Qual a diferença entre algoritmo e linguagem de programação?
O algoritmo é a lógica da solução de um problema. Já a linguagem de programação é a ferramenta utilizada para escrever essa solução de forma que o computador consiga entendê-la.

4. O que é JavaScript?
JavaScript é uma linguagem de programação muito utilizada no desenvolvimento web. Ela permite criar páginas interativas e também pode ser usada em servidores e aplicativos.

5. Onde o JavaScript pode ser utilizado atualmente?
O JavaScript pode ser utilizado em sites, aplicações web, aplicativos móveis, servidores, jogos e automações. Atualmente é uma das linguagens mais versáteis do mercado.

6. O que é o Node.js?
Node.js é um ambiente de execução que permite rodar código JavaScript fora do navegador. Ele é muito utilizado para criar servidores e aplicações back-end.

7. Qual é a função do Node.js no desenvolvimento com JavaScript?
A principal função do Node.js é permitir que o JavaScript seja usado no lado do servidor. Com ele é possível criar APIs, sistemas completos e automações.

8. O que é o VS Code?
O VS Code é um editor de código desenvolvido pela Microsoft. Ele oferece recursos que facilitam a escrita, organização e teste de programas.

9. Qual a diferença entre um editor de texto comum e um editor de código?
Um editor de texto comum serve apenas para escrever textos. Já um editor de código possui recursos específicos para programação, como destaque de sintaxe e sugestões automáticas.

10. O que é o terminal?
O terminal é uma ferramenta que permite executar comandos diretamente no sistema operacional. Ele é muito utilizado para rodar programas e gerenciar projetos.

11. Para que serve o comando node arquivo.js?
Esse comando serve para executar um arquivo JavaScript utilizando o Node.js. Assim, o código é interpretado e executado pelo computador.

12. O que é uma variável?
Uma variável é um espaço utilizado para armazenar informações dentro de um programa. Esses valores podem ser acessados e utilizados durante a execução do código.

13. Qual a diferença entre let e const?
A variável declarada com let pode ter seu valor alterado posteriormente. Já a declarada com const não pode receber uma nova atribuição após sua criação.

14. O que são tipos de dados? Cite exemplos.
Tipos de dados definem o tipo de informação armazenada em uma variável. Alguns exemplos são números, textos (strings), valores booleanos, objetos e arrays.

15. Qual a principal diferença entre JavaScript e TypeScript?
A principal diferença é que o TypeScript adiciona tipagem ao JavaScript, ajudando a identificar erros durante o desenvolvimento. Depois, o código TypeScript é convertido para JavaScript para ser executado.
