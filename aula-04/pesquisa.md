1. O que é uma estrutura de repetição?
É uma estrutura usada para repetir um comando ou bloco de código várias vezes.


2. Para que serve o laço for?
Serve para repetir uma ação quando a gente já sabe mais ou menos quantas vezes ela vai acontecer.


3. Para que serve o laço while?
Serve para repetir uma ação enquanto uma condição for verdadeira.


4. Para que serve o laço do-while?
Serve para repetir uma ação pelo menos uma vez e depois continuar enquanto a condição for verdadeira.


5. Qual é a diferença entre for, while e do-while?
for: usado quando já sabemos quantas repetições vamos fazer.
while: repete enquanto a condição for verdadeira.
do-while: faz o bloco pelo menos uma vez antes de verificar a condição.
